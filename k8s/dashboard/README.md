# Kubernetes Monitoring and Remediation Dashboard

A modern React-based dashboard for monitoring Kubernetes clusters, detecting anomalies, and providing remediation recommendations.

## Features

- Real-time cluster metrics visualization (CPU, Memory, Pod Status)
- Anomaly detection with confidence scoring
- Interactive remediation workflow
- AI-powered insights
- Cluster topology visualization
- Remediation history tracking

## Tech Stack

- **Frontend**: React.js, Next.js, Tailwind CSS
- **Visualization**: D3.js, React-Three-Fiber
- **API Communication**: Axios
- **Containerization**: Docker
- **Deployment**: Kubernetes

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Docker (for containerization)
- Kubernetes cluster (for deployment)

### Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
